## 💻 Pré-requisitos

* Java 11 ou superior
* Maven
* IDE favorita

## 💻 Tecnologias

* Java 11
* Spring 
  * Spring Boot
  * Hibernate Validation
  * DevTools
* Lombok
* MySQL 
* Flyway
* ModelMapper

## 🚀 Execução do projeto

```bash
# Clone do repositório
$ git clone https://github.com/ImGabreuw/mergulho-spring-rest
```

```bash
# Limpar o diretório target
$ mvn clean

# Criar o fat JAR
$ mvn package
```

```bash
# Entrar no diretório do projeto
$ cd algalog-api

# Entrar no diretório target
$ cd target/ 
```

```bash
# Executar o fat JAR
$ java -jar algalog-api-0.0.1-SNAPSHOT
```
